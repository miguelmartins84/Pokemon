Pokémon
===============================

This app is a Pokédex that allows users to list, search and save a list of their favorite Pokémons.

Main specs of this App:
* VIPER Architecture
* Swift with UIKit
* REALM database
* Firebase authentication and storing of the token
* Unit Tests
* UITests
